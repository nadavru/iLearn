In the context of ML, optimization is an iterative process of attempting to (typically) minimize a real valued function. Real valued functions come into play in ML as loss functions – functions that assess the performance of whatever ML construct (classifiers, self-driving cars, etc.) we want to create.




For example, let's look at the perceptron classifier. Perceptron is an early ML classifier whose operation is demonstrated in the attached illustration.

     • We are given a set of points on the plane (can be generalized to any dimension) – red and blue.

     • We want to find a line w such

           -Red points are below w.

           -Blue points are above w.


All these conditions are expressed mathematically in the illustration.

The initial set of labeled (red/blue) examples – the training set – is used to obtain w. Once we have w, any new data that we will pass through our classifier will be labeled as blue or red based on its relation to w.

Let's break down this expression:

      •The elements (x_1,y_1 ),…(x_n,y_n ) represent our training set.

           - Each x_i represents a point/vector (not a vector component)

           - Each y_i represents a label.

                •Blue/red, coded as +1/-1.

      •In general, we will classify data based on the value of the product p=w^T⋅x_i

           -For p>0, the x_i is above w and we classify the example as blue (see illustration).

           -For p"<"0 we classify the point as red (for p=0 we are undecided, assume it doesn’t happen).

                •I[mistake on〖 x〗_i ] is an indicator function: returns 1 for false classification, else 0.

           -Effectively that means our sum will count only false classifications.

      •Now, look at the overall expression:

           -If a blue example is classified as red: y_i=+1, w_T⋅x_i"<"0. So, we get:

                y_i⋅(w^T⋅x_i ) "<" 0

           We want our loss function to increase following a misclassification, so we need to reverse the sign – that leads to the final expression we see in the sum.

           -As 'homework', the reader is encouraged to work out how the same expression applies correctly when a red is classified as blue.

The L(w) we now have is a real valued function of two variables and can be optimized as such. A w for which L decreases represents an improved classifier over the training set.
